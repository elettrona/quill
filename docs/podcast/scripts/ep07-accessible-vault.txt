Linked note-taking. The whole personal knowledge movement, notes that link to each other, where ideas connect into a web you can explore. [LIAM]
Foundations first. A vault is just a folder of Markdown files. That's it. Tools, Vault, Open Vault, point at a folder, and QUILL indexes it and announces: vault name, forty notes, one hundred twelve links. There's a small cache file you can delete without losing a word. No database, no lock-in. Plain files you could open in Notepad.

[JESSICA]
Links are typed as double square brackets around a note's title. Type them anywhere in a note. Put your cursor on one and run Follow Wikilink, and you're in that note. Link to a specific heading with a hash, even to a specific block. If the note doesn't exist yet, QUILL offers to create it. If the name is ambiguous, you get a spoken chooser, never a guess.

[LIAM]
And there's completion: Complete Link or Tag at Cursor pops a filtered, spoken list as you're typing a link or a tag name, the screen-reader-friendly answer to those floating autocomplete windows that visual apps use.

[PAUSE]

[JESSICA]
Now the crown jewel: backlinks. Stand in any note and run Show Backlinks. QUILL answers, five notes link here, and lists them, each one read with the actual sentence its link lives in. Press enter on one and you land in that source note, at the mention.

[LIAM]
That's the graph view, spoken.

[JESSICA]
It goes further. Note Neighborhood shows everything one hop away in both directions, what this links to, what links to this. And Unlinked Mentions finds places where you wrote a note's name without linking it, so you can stitch your web tighter.

[LIAM]
Navigation across the whole vault is fast: Go to Note is a type-to-filter switcher. Search Vault searches every note, with regular expressions and whole-word options, and opens a result at its exact line. Show Tags is a tag pane where nested tags roll up, ask for the area tag and you get everything underneath it.

Daily notes: Open Today's Note creates or opens a dated journal entry, and Previous and Next Daily Note walk your journal like turning pages.

# QUILL Tutorials

Short, hands-on walkthroughs. Each one takes ten to twenty minutes at the
keyboard and teaches a complete, real workflow. They complement the
[user guide](../user%20guide/userguide.md) (the full reference) and
[The QUILL Cast](../podcast/README.md) (the audio course that covers the same
ground by ear).

1. [Your first hour with QUILL](01-your-first-hour.md) — install, profiles,
   the command palette, F1, and the habits that make everything else easy.
2. [Own the keyboard](02-own-the-keyboard.md) — the QUILL key, Quick Nav,
   rebinding keys, and the Spoken Echo.
3. [Rescue a scanned PDF](03-rescue-a-scanned-pdf.md) — the free-first
   Import / Convert Document tool and on-device OCR, end to end.
4. [From document to published audiobook](04-document-to-audiobook.md) —
   voices, Read Aloud, and the Audio Studio's three journeys: narrate
   documents, combine recordings, fix chapters by ear in the Workbench, and
   publish the feed — with incremental rebuilds, voice casting, AI chapter
   titles, and DAISY.
5. [Start an Accessible Vault](05-start-a-vault.md) — linked notes,
   backlinks, tags, daily notes, and Git sync, from an empty folder.
6. [Make a document accessible with GLOW](06-make-it-accessible-with-glow.md)
   — audit, fix with a compare, and grade the exported Word file.
7. [Write math without learning a new language](07-type-math-like-a-pro.md)
   — a ready-made formula gallery, typing an equation with ordinary keyboard
   characters, exploring one by ear, rendering live, and a real, editable
   Word equation on export and back.
8. [GitHub inside QUILL](08-github-inside-quill.md) — browse, open, and
   save files; work issues and pull requests; administer a repository
   (create, fork, rename, branch protection); and resolve merge conflicts
   and run an interactive rebase without ever leaving QUILL.
9. [Internet Radio](09-internet-radio.md) — find a station, save
   favorites, add your own custom streams, and control playback from the
   status bar, the tray, or the keyboard while you keep writing.
10. [Podcasts](10-podcasts.md) — subscribe, organize into folders,
    download with two independent pause controls, and play with per-show
    speed and resume-where-you-left-off, all without leaving QUILL.

Every tutorial assumes only a default QUILL install and a screen reader (or
none — everything here works identically by sight).

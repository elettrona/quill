QUILL for All 0.9.0 Beta 3 — Windows Installer
Publisher: Community Access

Thank you for installing Quill.

WHERE YOUR DATA LIVES
Quill stores settings, autosaves, dictionaries, and session data in:
  %APPDATA%\Quill

This folder is separate from the install directory. On uninstall you are
asked whether to remove it; upgrades never touch it automatically.

GETTING STARTED
  * Launch Quill from the Start Menu or the Desktop shortcut (if you chose one).
  * Press F1 or open Help > User Guide for the full guided manual.
  * An onboarding flow runs on first launch to introduce key features.

OPTIONAL TOOLS
Tools selected during setup (DECtalk, eSpeak-NG, Piper TTS, Pandoc, Braille Pack)
are bundled inside the install directory and require no separate installation.
To add or remove tools, re-run this installer and choose Modify.

PORTABLE EDITION
A portable build (no installer, runs from a USB stick or managed machine) is
available from the Quill releases page on GitHub.
